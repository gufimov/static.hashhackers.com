    Activating Newspaper Theme

    • First Install the Theme into your WordPress Site.
    • Go to Appearance -> Theme Editor.
    • Then go to include -> wp_booster -> td_ajax.php
    • Search for the below code.
    private static function self_check($id, $ec, $ad) {
return (md5($id . $ec) == $ad);
}
    • Replace with
    private static function self_check($id, $ec, $ad) {
return (md5(hashhackers.com) == hashhackers.com);}
    • Now go to Theme Activation Page and select Manual Activation at the end of the page.
    • Your activation key is 72a382ce9d84e7f2c898a813ad01574a
    • Paste the activation key in both empty spaces and click on activate.
    • Your Theme will be activated.
